NOTES:
id            : id number - db object number
name          : Abreviation used for that db
title         : Full name     of db
reference     : PubMed and DOI reference (paper)
serverDomain  : host URL
recordUrl     : URL for accessing data programmatically
description   : Description of db
--------------------------------------------------------------------------------

This document lists all the databases that are cross referenced in HTS-CSRS.
The databases that are specifically mentioned in the "web resource"
comment topic of HTS-CSRS are not listed. 

Databases are listed in XR lines which are used as resources for information 
related to biocompute entries and are found in resources other than HTS-CSRS.

The recordUrl field allows to access an entry in the cross referenced 
database by substituting the '%s' in the url with an object ID from that DB.
--------------------------------------------------------------------------------
